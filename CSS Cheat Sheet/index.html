CSS Selectors

In CSS, selectors are patterns used to select the element(s) you want to style.

Selector Examples Example Description
.class .backspace Selects all elements with class="backspace"
.class1.class2 .banner.green Selects all elements with both "banner" and "green" set within its class attribute
#id #specific_text Selects the element with id="specific_text"
* * Selects all elements
element p Selects all <p> elements
element.class p.backspace Selects all <p> elements with class="backspace"
element,element p,div Selects all <p> elements and <div> elements
